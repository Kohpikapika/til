# DenoでTask APIを作った

## 学んだこと

### 1. TypeScriptの型はruntimeでは保証されない

```ts
const body = await req.json() as CreateTaskRequest;
```

---

### 2. runtimeバリデーションが必要
```ts
if (typeof body.title !== "string") {
  return new Response("Invalid title", { status: 400 });
}
```

---

### 3. PowerShellは400を例外として扱う
Invoke-WebRequestは200系以外で例外を投げる。

try/catchでResponseを取り出す必要がある。

---

## 手順
### 1. サーバを起動
```bash
deno run --allow-net main.ts
```

### 2. タスクを追加（POST）
```powershell
Invoke-RestMethod -Method POST `
  -Uri "http://localhost:8000/tasks" `
  -ContentType "application/json" `
  -Body '{"title":"TypeScript勉強する"}'
```

### 3. タスクを確認（GET）
```bash
curl http://localhost:8000/memos
```

### 4. ステータスコードを含むレスポンスの情報を確認
```powershell
$response = Invoke-WebRequest -Method POST `
  -Uri "http://localhost:8000/tasks" `
  -ContentType "application/json" `
  -Body '{"title":123}' `
  -SkipHttpErrorCheck

$response.StatusCode
$response.StatusDescription
$response.Content
```
