# DenoでTask APIを作った

## 学んだこと

### 1. TypeScriptの型はruntimeでは保証されない

```ts
const body = await req.json() as CreateTaskRequest;
```

---

### 2. runtimeバリデーションが必要
```ts
if (typeof body.title !== "string") {
  return new Response("Invalid title", { status: 400 });
}
```

---

### 3. PowerShellは400を例外として扱う
Invoke-WebRequestは200系以外で例外を投げる。

try/catchでResponseを取り出す必要がある。

---
